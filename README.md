## 說明

riscv2os 是一本手把手帶你學習 RISC-V 到可以自製 RISC-V 處理器上的作業系統的電子書。

讀者可以透過這個學習資源，了解 C 語言是為何而生。

## 連結

本電子書使用 Github wiki 進行維護，連結如下:

[中文電子書 -- 從 RISC-V 處理器到 UNIX 作業系統](https://github.com/riscv2os/riscv2os/wiki)
